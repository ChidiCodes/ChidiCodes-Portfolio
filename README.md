# Divnex Chidi_Codes Portfolio Template

## Features
- Fully responsive on all devices  
- Built with HTML, CSS/BootStrap, JS 
- Modern UI and smooth animations  
- Clean code structure with comments  
- Fast loading  
- SEO-friendly  
- Editable project, skills, and contact sections  
- Ready for deployment (Netlify, Vercel, GitHub Pages)

---


## How to Edit
1. Open the **index.html** file with VS Code or any editor.  
2. Replace the content in:  
   - index.html
3. Customize colors inside `css/style.css`.  
4. Add your images inside the `images/` folder.

---

## Deployment
You can deploy this template for free using:

- **Netlify**
- **Vercel**
- **GitHub Pages**
- **Any shared hosting**

Just upload the folder or drag your project into Netlify.
